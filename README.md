# Crypto App

Android-приложение для отслеживания актуальных курсов криптовалют в реальном времени.

Данные загружаются с CryptoCompare API и кэшируются локально.

# Стек технологий

Kotlin

MVVM + Clean Architecture — разделение на слои data, domain, presentation

Dagger 2 

Room 

Retrofit 2 + Gson 

WorkManager 

LiveData 

ViewBinding 

Picasso 

RecyclerView + ListAdapter + DiffUtil 

Адаптивный UI
